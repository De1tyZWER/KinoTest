Простой веб-сайт онлайн-кинотеатра, созданный в рамках учебного проекта, реализующий регистрацию, авторизацию, личный кабинет и просмотр фильмов.

Технологии
Frontend: HTML, CSS, JavaScript
Backend: PHP
База данных: MySQL
Среда разработки: VSCode, OS Panel, PhpMyAdmin
  
Установка и использование

1. Клонируйте репозиторий
2. Запустите локальный сервер (например, OS Panel)
3. Импортируйте базу данных из папки app/kinodb.sql
4. Запустите сайт через локальный сервер

Настройка подключения к базе данных
В папке app/config.php укажите свои данные для подключения к базе данных:
<?php
return [
    "DB_HOST" => "localhost",
    "DB_NAME" => "kinoDB",
    "DB_USER" => "root",
    "DB_PASS" => "",
]
?>
