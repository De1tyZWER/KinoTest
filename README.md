Онлайн-кинотеатр
Простой веб-сайт онлайн-кинотеатра, созданный в рамках учебного проекта. Реализует регистрацию, авторизацию, личный кабинет и просмотр фильмов.

Технологии:
  Frontend: HTML, CSS, JavaScript
  Backend: PHP
  База данных: MySQL
  Среда разработки: VSCode, OS Panel, PhpMyAdmin
  
Установка и использование
1. Клонируйте репозиторий
2. Запустите локальный сервер (например, OS Panel)
3. Импортируйте базу данных из папки database/
4. Запустите сайт через локальный сервер
